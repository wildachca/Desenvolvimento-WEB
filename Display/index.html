1. Display: block

Os elemento com display: block ocupam toda a largura disponivel e sempre comecam em uma nova linha.

caixa block 1
caixa block 2
caixa block 3

2. Display: inline

Elementos 'inline' so ocupa o espaço do seu conteudo e ficao lado a lado. Eles "iguara" as dificeis.

inline A inline B (Com conteudo maior) inline C

3. Display: inline-block

O melhor dos dois mundos: fica lado a lado E respeita as definiçoes de "width" e "height"

inline-block I
inline-block II
inline-block III
